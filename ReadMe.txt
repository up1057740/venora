________About the software__________ 

This software is written in html,css,JavaScript,Nodejs, uses MySQL module to connect to the desired SQL Database 
and requires a user to Install phpMyAdmin and xampp. 
-admin info: mail:bitsori.danae@gmail.com password:danae.bitsori
_Pre-requisites_ 

1 - Install express-session of Nodejs
2 - Install phpMyAdmin from this link https://www.phpmyadmin.net/downloads/ . Create a schema with the name 'venora' and import 
sql file 'venora.sql'
3 - Install xampp from this link https://www.apachefriends.org/index.html

	
________Features_____________________

1 - Works on Windows.(Tested and ran successfully) 
2 - Multiple features for different sessions
3 - Shows the process output in the console area. 
5 - Its a free software
